# es2015-find-field-game

Simple game written in ES2015 with webpack, babel & jasmine

Demo:
http://marekstabla.pl/es2015-find-field-game/

Contributors:
Marek Stabla

### Installation

* Install Node.js
* Install webpack
  * ```npm install -g webpack```
* Clone repository, and navigate to it
* Install requirements
  * ```npm install```
* Build or test app
  * Build: ```webpack```
  * Test: ```npm tests```
